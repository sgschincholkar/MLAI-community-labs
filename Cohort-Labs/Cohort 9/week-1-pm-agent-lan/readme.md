# Week 1: Build Your First AI Agent

![banner](./assets/banner.png)

Welcome to Week 1 of the AI PM Certification.

This week you go from zero to a fully working AI-powered contract review product — built by you, end to end. No prior coding experience needed. Each lab builds on the last, so work through them in order.

By the end of this week, you'll have:

✦ A live AI agent that reads contracts and answers questions about them
✦ A working web app prototype built with Claude Code
✦ The two connected via a webhook into one real product

---

## Labs

### [Lab 1.1 — Build Your First AI Agent in n8n](./1.1%20-%20n8n-agent/readme.md)

You'll set up an AI agent inside n8n using a pre-built workflow, connect it to OpenAI's GPT-5-mini model, and test it by uploading a contract and asking questions about it. This is the intelligence layer — the brain of the product.

**What you'll learn:** What an AI agent is, how system messages and user messages work, how to configure an OpenAI model, and how to run your first agent end-to-end.

**Time:** ~25 minutes

---

### [Lab 1.2 — Build Your First Prototype with Claude Code](./1.2%20-%20claude-prototype/readme.md)

You'll use Claude Code to build a contract review web app in two iterations — first a working shell with simulated responses, then a polished UI with a live contract preview panel. No code written by hand.

**What you'll learn:** The difference between Claude Chat, Claude Cowork, and Claude Code. What CLAUDE.md is and how to use it. How to go from a prompt to a testable prototype in one session.

**Time:** ~25 minutes

---

### [Lab 1.3 — Connect Your n8n Agent to Your Prototype](./1.3%20-%20connect-n8n-prototype/readme.md)

You'll replace the dummy responses in your prototype with real AI answers — by wiring up the n8n agent from Lab 1.1 to the web app from Lab 1.2 using a webhook. By the end, you have one fully working product.

**What you'll learn:** What a webhook is, how to configure one in n8n, how CORS works, and how to connect an external frontend to an AI agent backend.

**Time:** ~30 minutes

---

> ✓ Tip. If you get stuck in any lab, each one has troubleshooting steps built in. Read the error carefully — most issues in these labs have a one-step fix.
