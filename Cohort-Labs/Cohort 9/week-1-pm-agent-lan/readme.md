# Week 1: Build Your First AI Agent

![banner](./assets/banner.png)

Welcome to Week 1 of the AI PM Certification.

This week, you go from zero to a fully working AI-powered contract review product — built by you, end to end. Not a tutorial you follow passively. Something you actually ship.

Each lab builds on the last, so work through them in order. By the end of the week, you'll have three things that connect into one real product: an AI agent that reads contracts, a web app that looks like something you'd show a stakeholder, and the two wired together into a live experience.

No prior coding experience needed. Just curiosity and about 80 minutes.

---

## The Labs

### [Lab 1.1 — Build Your First AI Agent in n8n](./1.1%20-%20n8n-agent/readme.md)

This is where you build the brain. You'll set up an AI agent inside n8n — the thing that actually reads the contract and answers questions about it. You'll connect it to OpenAI, write the system message that defines how it behaves, and test it live by uploading a real contract and asking questions in plain language.

By the end, you'll understand what an AI agent actually is under the hood — not the buzzword version, the real one.

**Time:** ~25 minutes

---

### [Lab 1.2 — Build Your First Prototype with Claude Code](./1.2%20-%20claude-prototype/readme.md)

This is where you build the face. You'll use Claude Code to generate a contract review web app — an upload area, a chat panel, a two-column layout that looks like a real product. No code written by hand.

But there's a twist. Midway through, you'll hit a moment of friction: the app works, but it doesn't feel like *yours*. That friction is the setup for the most important concept in this lab — CLAUDE.md. You'll discover it at exactly the right moment.

**Time:** ~30 minutes

---

### [Lab 1.3 — Connect Your n8n Agent to Your Prototype](./1.3%20-%20connect-n8n-prototype/readme.md)

This is where the two come together. You'll replace the dummy responses in your web app with real AI answers — by wiring the n8n agent from Lab 1.1 directly into the prototype from Lab 1.2 using a webhook. One message in the chat, one real response from the agent.

By the end, you have a working product. Not a demo. Not a mockup. Something you built.

**Time:** ~30 minutes

---

> If you get stuck in any lab, each one has troubleshooting guidance built in. Read the error carefully — most issues in these labs have a one-line fix.
